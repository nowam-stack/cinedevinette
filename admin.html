📤 Mettre votre site à jour

🎬 Étape 1 — Exporter votre configuration

Cliquez sur le bouton ci-dessous pour télécharger votre liste de films mise à jour. Ce fichier s'appelle films.json.

📁 Étape 2 — Uploader sur GitHub

Une fois le fichier téléchargé, suivez ces étapes :

  1. Ouvrez votre dépôt sur github.com
  2. Cliquez sur "Add file" → "Upload files"
  3. Glissez-déposez films.json
  4. Cliquez "Commit changes"
  5. Attendez 2 minutes que GitHub mette à jour votre site

✅ Étape 3 — Vérifier

Ouvrez votre site en navigation privée pour voir les changements. Votre film du jour doit apparaître avec les nouveaux indices.

💡 Astuce — Importer si vous changez d'appareil

Vos données sont sauvegardées dans ce navigateur. Si vous changez d'ordinateur, utilisez "Importer" dans l'en-tête pour recharger votre films.json.
